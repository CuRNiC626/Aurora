# Aurora Engine

Лёгкий и расширяемый движок, ориентированный на .NET Framework 4.8, с упором на производительность, простоту и контроль над низкоуровневыми структурами.

## 📌 О проекте

Aurora — это кастомный движок, разрабатываемый с нуля. Основная цель — создать удобную и эффективную основу для разработки приложений/игр с минимальными зависимостями.

Проект ориентирован на:
- Высокую производительность
- Минимальный overhead
- Контроль над памятью и структурами данных
- Простую архитектуру

## ⚙️ Технологии

- .NET Framework 4.8
- C#
- Без внешних зависимостей (по возможности)
